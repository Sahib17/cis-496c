main.tsx -> never touch, entry point

App.tsx -> Routes, global layout, context providers, top level structure go here, No business logic

Pages/ -> route level components, each file = one page, example 
                pages/
                    Home.jsx
                    Login.jsx
                    Dashboard.jsx

components/ -> reusable UI pieces, example
                components/
                    Navbar.jsx
                    Button.jsx
                    Card.jsx
                    Modal.jsx

hooks/ -> reusable logic in components, no UI
                hooks/
                    useAuth.js
                    useFetch.js
                    useDebounce.js

context/ -> global state management
                context/
                    AuthContext.jsx
                    ThemeContext.jsx

services/ -> API calls go here
                services/
                    api.js
                    authService.js
                    userService.js

utils/ -> Helper functions
                utils/
                    formatDate.js
                    calculateTotal.js


UI piece → components
Route page → pages
Shared logic → hooks
Global state → context
API → services
Pure helper → utils
